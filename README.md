# 🌾 FarmWise Connect – Smart Agriculture Platform

FarmWise Connect is a Smart Agriculture / Farmer Support Website designed to help farmers make better farming decisions using technology.  
The platform provides crop advisory, weather insights, market price information, and awareness about government schemes.

---

## 🚀 Live Demo
👉 https://farmwise-connect.vercel.app/

---

## 📌 Features
- 🌱 Crop advisory based on season and soil
- 🌦️ Weather forecast for better planning
- 💰 Market (mandi/APMC) price information
- 🏛️ Government agriculture schemes
- 📱 Fully responsive, mobile-friendly UI
- 🌍 Clean and farmer-friendly design

---

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Version Control:** Git & GitHub
